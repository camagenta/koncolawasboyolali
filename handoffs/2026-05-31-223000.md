## Overflow Fix (Root Cause + Solution)

- **Root cause**: `app-shell.tsx:88` — div `flex-1 flex-col` inside flex container. Default CSS `min-width: auto` prevents flex items from shrinking below content intrinsic width. At 375px viewport, container forced to 551px.
- **Fix**: Added `min-w-0 overflow-x-hidden` to the flex wrapper so flex item can shrink to parent width.
- **Pagination**: Changed from `overflow-x-auto > w-max` to `flex-wrap` so page buttons wrap naturally.
- **Verification**: All 5 pages (alumni, sukses, jobs, bisnis, alumni-mengajar) pass — docScrollWidth == 375px at 375px viewport. Also verified at 320px. Bottom sheet renders correctly.
